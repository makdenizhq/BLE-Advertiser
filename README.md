# BLE Advertiser App

This is a simple Android application that advertises a custom BLE (Bluetooth Low Energy) signal. The signal contains an encrypted payload with a user-defined name and message. The encryption key can be a fixed key or a daily-changing key.

## How to use the application

1.  Enter a name and a message.
2.  Choose between a fixed key or a daily-changing key.
3.  If you choose a fixed key, you can edit the default key.
4.  You can share the encryption key via a QR code.
5.  Click on "Start Advertising" to start the BLE advertisement.
6.  Click on "Stop Advertising" to stop the BLE advertisement.

## How to build the application from source

1.  Clone this repository.
2.  Open the project in Android Studio.
3.  Build the project.
